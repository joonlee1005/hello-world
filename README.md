## 줄거리

실존 인물인 호주 피아니스트 데이비드 헬프갓(David Helfgott)의 생애를 바탕으로 한 전기 영화다. 천재적 재능을 지닌 데이비드는 권위적인 아버지의 억압 속에서 성장하고, 영국 왕립음악원 유학 중 라흐마니노프 피아노 협주곡 3번을 완주한 직후 정신적으로 붕괴하여 조현병 진단을 받는다. 수십 년의 공백과 방랑 끝에 한 여성의 사랑으로 다시 무대에 오르기까지의 여정을 그린다.

## 음악이 작품 속 질병을 묘사하는 방식에 관한 단상

《샤인》에서 라흐마니노프 협주곡 3번은 단순한 배경음악이 아니라 데이비드의 내면 상태를 직접적으로 가시화하는 매개체다. 이 곡의 압도적인 기술적 요구는 주인공이 감당해야 할 심리적 부담과 겹쳐지며, 연주의 완성과 정신의 붕괴가 동시에 일어나는 장면은 예술적 성취와 정신적 한계가 얼마나 가깝게 맞닿아 있는지를 보여준다. 영화는 음악을 치유의 도구이자 파괴의 촉매로 동시에 제시함으로써, 조현병을 단순한 비극으로 낭만화하지 않고 음악과 광기, 천재성과 고통이 뒤얽힌 복잡한 인간 경험으로 묘사한다.
